# Ghost Planner

## Prerequisites

- Python 3.9
- npm 8.1.0
- node v17.0.1

## Installation

`cd planner-fast-api/`

`npm i`

`python -m venv venv`

MacOS/Unix `source venv/bin/activate`

Windows `venv\Scripts\Activate`

`pip install -r requirements.txt`

## Running the App

In the server folder run `uvicorn server.main:app --reload`

In the root folder run `npm run serve`
